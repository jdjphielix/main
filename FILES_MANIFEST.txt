QUICK START
================================================================================

1. Copy files to your React project
2. Ensure Tailwind CSS 3.x is configured
3. Add custom animation to tailwind.config.js:
   animation: {
     'slide-up': 'slideUp 0.3s ease-out',
   },
   keyframes: {
     slideUp: {
       '0%': { transform: 'translateY(100%)', opacity: '0' },
       '100%': { transform: 'translateY(0)', opacity: '1' },
     },
   }
4. Import in router: <Route path="/leads" element={<LeadsPage />} />
5. Done! Explore the UI with mock data

For backend integration, see INTEGRATION_GUIDE.md
